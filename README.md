## ‚ú® Key Features

* **Smart SKU Management:** Efficiently add and search products using unique custom codes (e.g., `123ab`, `190#23`).
* **Dynamic POS Terminal:** Real-time billing system with an integrated **Cash Return Calculator**.
* **Sales Analytics:** Interactive line charts visualizing sales performance for the last 7 days using **Chart.js**.
* **Inventory Control:** Complete CRUD functionality (Add, View, Delete) for products with automated stock valuation.
* **Mobile Optimized:** Fully responsive UI designed to work perfectly on mobile browsers and **Termux**.
* **Professional Icons:** Clean interface utilizing **FontAwesome 6** icons.
